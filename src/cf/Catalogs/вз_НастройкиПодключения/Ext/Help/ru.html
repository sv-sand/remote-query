Содержит описание настроек публикации внешней базы данных.

Описание доступных полей:


Наименование - представление настройки.

Идентификатор - уникальный строковый реквизит, для выбора настройки во внешней базе при создании внешнего запроса.

Вид подключения - разновидность сервиса для подключения к внешней базе данных.

Версия протокола - версия протокола, поддерживаемая внешней базой данных. Можно узнать версию открыв окно настроек библиотеки интеграции (Настройки (вз)).

Строка подключения - строка для подключения к сервису

Имя пользователя - имя пользователя для авторизации в сервисе

Пароль - пароль для авторизации в сервисе
