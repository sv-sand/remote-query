Справочник предназначен для хранения текстов запросов или исполняемого кода, используемых при обработке внешнего запроса.

Описание доступных полей:


Идентификатор - идентификатор внешнего запроса, уникальный строковый реквизит, который указывается во внешней базе.

Вид - определяет вид операции выполняемой внешним запросом: Запрос 1С или Исполняемый код 1с.

Текст запроса - хранит текст запроса 1С, который будет выполнен при обработке внешнего запроса. Доступен только с видом "Запрос".

Исполняемый код - хранит программный код 1С, который будет исполнен при обработке внешнего запроса. Доступен только с видом "Код". 

Результат исполнения кода должен быть установлен в переменную Результат, далее значение переменной будет сериализовано и возвращено во внешнюю базу, которая инициировала запрос. Значение переменной результат может быть любым сериализуемым типом, а также таблицей значений, которая при передаче будет преобразована в массив структур.

В исполняемом коде можно использовать Параметры принятые из внешнего запроса. Пример кода: 

Результат = КадровыеДанные.ПолучитьКадровуюИсториюПоСотруднику(Параметры.ФизическоеЛицо);

Параметры - коллекция параметров, которые можно использовать в запросе. 

Если установлен флаг Внешний, параметр обязательно должен быть во внешнем запросе, иначе параметр указывается в текущей настройке из данных текущей базы.

При нажатии на кнопку Заполнить программа найдет запрос 1С в исполняемом коде и заполнит параметры из запроса.
